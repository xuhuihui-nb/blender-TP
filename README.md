# blender-TP
手动拓扑/Manual topology

目前的基础功能:

1鼠标绘制拓扑线

2创建环绕线

3自动填充栅格(此功能只有简单的填充,复杂的算法还在开发中）

<img width="344" height="176" alt="image" src="https://github.com/user-attachments/assets/f1dbfdae-2c62-4895-997a-2b789550bb21" />

安装方法：将解压后的文件夹内的名为TP的文件夹移动到你的blender插件存放的文件夹内.默认的blender zip文件 安装方式是无效的

Installation method: Move the folder named TP from the decompressed folder to the folder where your Blender plugin is stored The default blender zip file installation method is invalid

1. Use the mouse to draw topological lines
