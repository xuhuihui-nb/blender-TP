# blender-TP
手动拓扑/Manual topology

这个插件可以用来提高手动拓扑的效率，核心方式是通过对绘制方式的优化来辅助画结构线。

你只要专注于绘制结构线，其它的由插件帮你完成。这个件拓扑出来的内容支持在retopoflow中使用松弛调整工具对结果进行整体优化。

插件内包含有详细的中英文教程

安装方法：将解压后的文件夹内的名为TP的文件夹移动到你的blender插件存放的文件夹内.默认的blender zip文件 安装方式是无效的

只需简单的绘制一个圈，就能有不错的拓扑效果。
